# MARBLE: Masonry and AI-driven Reconstruction of Buildings for Lifecycle Evaluation

A software for imaging your masonry building (LOD, DADT, FEM) and assessing its mechanical performance.

Integration of:

https://github.com/eesd-epfl/LOD3_buildings

https://github.com/eesd-epfl/FEM_buildings

https://github.com/eesd-epfl/DADT_buildings

https://github.com/eesd-epfl/topo_crack_detection

https://github.com/eesd-epfl/crack_kinematics
